Hyperledger Composer Quickstart

Proof of Concept quickstart template that can be used to quickly develop a web application on top of Hyperledger Composer and deploy it in a fully contained, standalone environment.

LEARN MORE LIVE DEMO

What does it do

Quickstart template allowing you to develop a Hyperldeger Composer network and a web application to visualise the use case, while taking care of the deployment in a Fabric network.

Step 1: Define the Hyperledger Composer business network by filling in the Data model (.cto), access control rules (.acl) and transaction logic files (.js). Normally for a proof of concept this would also imply defining multiple participant types (as assets) and adding some initial accounts.

Step 2: Update the sample Angular frontend application to support the account types and business logic implemented. Angular developmente experience is required for this.

Optionally the landing page document (this page) can also be updated with a description of the project.

Step 3: Deploy both the web application and blockchain network either on a local development machine or a production-ready server. Deployment can be easily done cross-platform via Docker and Docker-compose.

Benefits

Using this quickstart template allows you to focus on writing the business relevant blockchain code instead of focusing on deployment or library version management.

Ease of deployment

Out of the box deployment of the correct verisons of Hyperledger Fabric network and Hyperledger Composer framework.

Quickstart webapplication

A web application is provided out of the box, allowing you to perform basic operations like user management and asset transfer.

Sample Hyperledger Composer code

Sample blockchain implementations are provided for basic use-cases, to showcase how real-world scenarios can be implemented in Hyperledger Composer.

Technical Features

Cross-platform technologies are used to enable implementing and deploying this Proof of Concept on multiple operating systems.

Angular 5 & Bootstrap 3

A quickstart web application with basic features pre-built.

Node Js

A Node Js application that manages the build of the composer code and deployment on top of a Hyperldeger Fabric network.

Docker & Docker-compose  

Container technology that encapsulates the web application, backend and full Hyperledger Fabric network, to allow easy deployment in any environment.
